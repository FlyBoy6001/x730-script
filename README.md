# x735-script
This is the safe shutdown script for x735;

How to use?

* step 1:
> wget https://raw.githubusercontent.com/geekworm-com/x730-script/master/x730.sh

> sudo chmod +x x735.sh

> sudo bash x735.sh

* step 2:

> printf "%s\\n" "alias x735off='sudo x735shutdown.sh'" >> ~/.bashrc

* step 3:
> sudo reboot
